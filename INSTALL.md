# Установка NovaReader

## Windows

### Простая установка
1. Скачайте `NovaReader_Windows_v1.0.zip`
2. Распакуйте архив в любую папку (например, `C:\Program Files\NovaReader`)
3. Запустите `NovaReader.exe`

### Создание ярлыка
- Щёлкните правой кнопкой мыши на `NovaReader.exe` → "Отправить" → "Рабочий стол (создать ярлык)"
