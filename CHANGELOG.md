# Changelog

## [1.0] - 2025-04-03

### Добавлено
- Первый публичный релиз NovaReader
- Поддержка форматов: FB2, EPUB, PDF, CBZ
- TTS с голосами Piper (Ирина, Денис, Дмитрий)
- Подсветка текста (4 цвета + 3 стиля)
- Закладки и заметки
- Сохранение прогресса чтения
- Настройка темы (светлая/тёмная/сепия)
- Настройка шрифта и межстрочного интервала
- Режим страниц (одна/две/авто)

### Исправлено
- Оптимизировано потребление памяти для больших FB2 книг (с 1.6 ГБ до 550 МБ)
- Исправлена проблема с пустой страницей при открытии книги
- Устранены зависания при переключении глав
- Починен прогрев WebEngine на Linux

### Известные проблемы
- На Linux с драйверами NVIDIA редкие замедления при первом запуске книги
